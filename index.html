📚 Game Dev Learning Library

This is a list of curated sites that can help you to develop coding or game making skills.

🧠 Learn by Looking at Code

  • GitHub Search: Website Templates – Find templates with open licenses.
  • HTML5 UP – Beautiful, responsive HTML templates (CC license).
  • Templatemo – Free HTML templates (with credit).
  • One Page Love – Curated single-page templates.
  • BootstrapMade – Bootstrap templates (free + paid).

🎨 Frontend Playground Sites

  • CodePen – Small interactive demos, often reusable with credit.
  • Glitch – Full-stack web apps, many open source.
  • GitHub Pages Showcase – Real user websites hosted via GitHub.
  • FreeFrontend – Free UI examples and code snippets.
  • CSS-Tricks – Articles and real code examples (great for learning).

🎮 Learn Game Development (Interactive)

  • CodeCombat – RPG game that teaches JavaScript/Python.
  • Twine – Open-source tool for making interactive stories.
  • Bitsy – Tiny narrative game engine.
  • PICO-8 – Retro game engine for learning Lua.
  • Tynker – Visual programming for teaching logic (great for kids too).

🔓 Open & Educational Resources

  • MDN Web Docs – Canvas, JavaScript, interactivity basics.
  • OpenGameArt.org – Free game assets and sprites.
  • GitHub Topic: game-dev – Public game projects you can learn from.
  • Itch.io – Open Source Games – Learn how indie games are made.

🕵️ Want to Make Your Code Harder to Copy?

  • Obfuscator.io – Makes JS hard to read (not secure, but annoying).
  • UglifyJS – Command-line tool to minify & obfuscate JS.

Note: These tools don’t *prevent* copying—just make it less fun.

The point is not for you to learn from all the site links here, rather to find one that you like ~
